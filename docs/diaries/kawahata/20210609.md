# Rasberry PiにVS Codeを入れる
TerminalでVS Codeをインストールする
```
sudo apt update
sudo apt install -y code
```
ホームボタン→プログラミングにVS Codeが出現する
VS Code内でCtrl+Shift+Pを押して`Settings Sync: Turn`と検索し、Turn onにする
