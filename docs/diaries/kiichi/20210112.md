# ルーティング：Route::resource, Route::group(['middleware' => ['auth']], function () {});, Redis

## Route::resource

ルーティングにRoute::resouceを指定することで、CRUDルーティングを一度に行うことができる。公式のドキュメントの対応表参照。

## Route::group(['middleware' => ['auth']], function () {});

ログインしていないとそれ以降のルートにアクセスできなくする。
