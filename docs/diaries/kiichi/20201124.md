### Gitの理解・実践

VSCodeでuser.nameとuser.emailを求められたとき、

`git config --global user.email "you@example.com"`

`git config --global user.name "Your Name"`

を実行する。
それでも解決しない場合は、--globalオプションを無くして実行するとうまくいく。
